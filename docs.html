System Layers

Visualizes the 8-layer architecture stack using the <cc-layer-stack> component:

  • Component Library: 69 shared web components
  • Design System: theme.css + landing.css + glass morphism
  • Database: Supabase with 63 tables, RLS, and Auth
  • Skills Engine: 37 automation skills and orchestrators
  • Cron System: 8 batch jobs + trigger_queue processor
  • Memory: 24 markdown files + DB dual-write
  • Runtime: OpenClaw agent with Claude and 20+ integrations
