The App Pipeline

  • AI receives request via chat or cron job
  • create-app skill orchestrates — delegates to page sub-skills
  • Sub-skills build each page: landing, ads, docs, ideas, logs, prompts
  • Each page uses shared components + theme.css automatically
  • Git init → push to last-rev-llc/ah-<app-name>
  • Auto-deployed to <name>.adam-harris.alphaclaw.app
  • App registered in memory/apps.md

Conversation → live website in ~2 minutes.

Design System — theme.css

1,600 lines powering all 44 apps from a single file.

  • Dark mode default with light mode toggle
  • Glass morphism — backdrop-filter blur + translucent cards
  • CSS custom properties for consistent theming
  • Responsive grid utilities and card styles
  • Gradient accents: amber → gold
  • System font stack with serif option
  • Built-in animation classes

Key Architectural Wins

  • Component-first: 40 shared web components = zero duplication
  • Convention over config: Same structure for every app
  • Memory continuity: 17 files + DB = agent never loses context
  • Automated quality: Nightly drift detection catches regressions
  • Sub-2-minute deploys: Conversation → live site
  • Self-documenting: Memory files auto-update as system grows
  • Loop prevention: Failing crons auto-disable before damage
  • Single theme: One CSS file, 44 apps, dark + light mode
